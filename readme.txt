== Description ==

WP1 Like is a simple Like button plugin for posts, pages, custom post types and WooCommerce products.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'WP1 Like'
3. Activate 'WP1 Like' from your Plugins page.
4. Visit 'Settings > WP1 LIke' and change settings according to your website theme.

= From WordPress.org =

1. Download `wp1-like` and extract the zip.
2. Upload `wp1-like` directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'WP1 Like' plugin through the 'Plugins' menu in WordPress
